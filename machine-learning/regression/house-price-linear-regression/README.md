# House Price Prediction using Linear Regression

## Objective
Build a simple linear regression model to predict house prices based on area.

## Steps
- Created synthetic dataset
- Performed train-test split
- Trained Linear Regression model
- Evaluated using MAE, MSE, R2 score
- Visualized predictions

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Key Learning
Understanding how linear regression models relationships between variables.
