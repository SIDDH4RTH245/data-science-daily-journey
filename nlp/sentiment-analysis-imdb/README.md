# IMDB Sentiment Analysis

## Objective
Build a sentiment classification model using NLP techniques.

## Steps
- Loaded IMDB review dataset
- Encoded sentiment labels
- Applied TF-IDF vectorization
- Trained Logistic Regression model
- Evaluated performance

## Tools Used
- Python
- Scikit-learn
- TF-IDF
- Logistic Regression

## Key Learning
Understanding text preprocessing and classification in NLP.
